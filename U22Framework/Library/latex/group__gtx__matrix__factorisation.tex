Performs RQ factorisation of a matrix. Returns 2 matrices, r and q, such that r is an upper triangular matrix, the rows of q are orthonormal and span the same subspace than those of the input matrix, and r$\ast$q=in. Note that in the context of RQ factorisation, the diagonal is seen as starting in the lower-\/right corner of the matrix, instead of the usual upper-\/left. Given an n-\/by-\/m input matrix, r has dimensions \mbox{\hyperlink{stb_8h_abb702d8b501669a23aa0ab3b281b9384}{min(n,m)}}-\/by-\/m, and q has dimensions n-\/by-\/min(n,m).
